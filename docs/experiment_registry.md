# Experiment Registry

Canonical model targets supported by the remake framework:

- `logreg`
- `rf`
- `xgb`
- `mlp`
- `mlp_torch`

Canonical training outputs:

- `config_used.json`
- `metrics_summary.json`
- `predictions.csv`
- `raw_log.txt`
- `artifacts_manifest.json`
- optional search/importance/history CSV files

Canonical analysis outputs:

- ablation tables
- temporal study tables
- calibration summaries
- aggregated seed summaries
- LaTeX-ready tables
- replay-level error summaries
